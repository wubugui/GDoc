# 探索与交互

这页讲清楚：场景里能碰的东西分几类、怎么调查和拾取、场景之间怎么切换。读完你就能把雾津的每条街摸透，不会漏掉调查点，也不会在转场时找不到门。

---

## 这是什么（30 秒看懂）

雾津由许多**场景**拼成：渡口、老街、义庄、城隍庙、土地庙、阎王岭外径……场景之间用门洞、台阶、渡口缆绳等连接。默认你在**探索状态**，用移动和互动键把地图摸透——这是全游戏里你待得最久的状态，对话、遭遇、小游戏都是从探索里「长」出来的分支。

打个比方：探索就像你真的拎着伞在雾津的湿巷子里溜达，走到哪个门口、哪个摊子、哪个人跟前，伞檐一抬眼就知道这地方有没有戏可看。

## 入门：新手怎么玩

### 场景里有什么可互动

走近某些位置，会出现互动提示。常见几类：

| 类型 | 你感受到的 | 例子（雾津） |
|---|---|---|
| **调查** | 查看物件、读说明、找线索 | 义庄棺盖上的纸封、渡口沉箱痕迹 |
| **拾取** | 捡起地上的东西进背包 | 湿了的鞋、掉落的符纸 |
| **对话** | 跟 NPC 开口 | 关二狗蹲处、李天狗在土地庙台阶 |
| **转场** | 切到另一张地图 | 进城隍庙山门、出城往阎王岭 |
| **遭遇** | 弹出选项页，紧要抉择 | 纸人巷对峙、庙前规矩判定 |
| **小游戏** | 进入糖画、扎纸、水域 | 庙会糖画摊、江边捞物 |

统一操作：站进提示范围，按 `E` 或 `空格`。

```mermaid
flowchart TD
  M[在场景移动] --> N{有互动提示?}
  N -->|否| M
  N -->|是| P[按 E / 空格]
  P --> Q[调查 / 对话 / 拾取 / 换场景 / 遭遇 / 小游戏]
  Q --> M
```

### 小例子：从土地庙到渡口，照着走一遍

新手第一次探索，可以按这个顺序练手：

1. 在**土地庙**外跟李天狗搭话，接到寻狗线索。
2. 调查庙旁供桌，拾取若有提示的符纸碎角。
3. 走转场到**渡口**，调查缆绳桩听闲话。
4. 若任务指向河边，找水域互动点尝试捞物（见 [小游戏玩法](./minigames)）。
5. 回头翻一眼**任务面板**和**规矩本**，确认这一路有没有新进展。

走完这五步，你就完整体验了「移动 → 互动 → 调查/拾取 → 转场 → 回顾进度」的一整套探索节奏，后面所有场景都是这套动作的变体。

## 进阶：玩深的技巧

### 调查与观察，讲究先后顺序

调查点往往给**见闻线索**，不是每次都有物品。雾津里：

- 码头听捞尸人闲话，可能解锁档案里的见闻。
- 城隍庙看香炉裂纹，后面规矩或遭遇会对上。
- 义庄纸人姿态不对，别手快——**先调查再决定动不动**。这不是随口一说：不少义庄相关的遭遇结果，取决于你调查过没有、规矩学到哪一层。

有些调查点**要任务或规矩到了才出现**；第一次路过没有提示，过阵子再来可能有。养成「同一场景多回头看看」的习惯，比一次性扫过更容易发现新热区。

### 拾取与背包

拾取后物品进**背包**。任务道具、消耗品、铜钱等类型不同，能不能用、在哪儿用，看物品说明和当前剧情。

- 李天狗让你找的引魂灯材料，通常得先捡到齐，才能在扎纸或对话里交差。
- 香烛、符纸之类，遭遇或险境里可能消耗；缺了记得去渡口货郎或城隍庙香烛铺看看 [物品与买卖](./items-shop)。

拾取类热区大多是**一次性**的：捡完这个点就不会再给东西了，所以「回头补漏」主要是为了见闻和调查，而不是指望同一个点反复刷物品。

### 场景切换

**转场**热区多在门、渡口、巷子口。走过去互动即可切换地图。

注意：

| 情况 | 说明 |
|---|---|
| 门打不开 | 剧情未到时锁着；或要先跟某人对话、拿钥匙 |
| 同地名不同内容 | 可能进了**另一位面**——见 [操作与界面 · 位面](./controls#位面与画面变化) |
| 阎王岭 | 城外险地，建议存档后再进 |
| 转场后 NPC 位置变了 | NPC 会随剧情或时段挪动摊位/岗位，不是原地不动的雕塑 |

### NPC 与走动

NPC 站在场景里，有的会动（巡逻、作揖），有的蹲着不动。互动距离不必贴脸，但离太远不会出对话。

关二狗、李天狗、庙祝、货郎、纸扎匠都会在固定或条件满足后出现。某人突然不在老位置，多半是剧情推走了，或当前位面看不见他——回想一下最近做了什么，而不是当成 bug。

### 任务与探索配合

打开**任务面板**看当前目标——写清要去哪、找谁、拿什么。任务说明和 [档案 · 见闻录](./archive) 可以对照着看，避免在老街空转。养成的习惯：卡关先看任务面板，再翻档案，最后才考虑要不要满地图重新扫一遍热区——这个顺序效率最高。

### 高效探索的小窍门

- **走过的场景别急着划掉**：热区可能随任务进度重新出现，第一次没提示不代表永远没有。
- **调查优先于动手**：义庄、纸人、供桌一类物件，先按调查键看反馈，再决定要不要进一步互动。
- **留意提示图标的差异**：调查、拾取、转场、对话、遭遇的提示外观通常不同，练出眼力能提前判断这个热区大概是什么类型，减少误触。

## 常见问题

| 现象 | 多半原因 |
|---|---|
| 站在物件旁边没提示 | 没站对触发范围，或该热区还没满足任务/规矩条件 |
| 调查过的地方东西不见了 | 一次性热区调查/拾取后会消失或变化，正常现象 |
| 同一地点突然内容不一样 | 可能进入了不同**位面**，人和物都可能变 |
| 转场门口按了没反应 | 门被剧情锁住，回头看任务提示或该去找谁问一句 |
| 东西没捡到，会不会卡关 | 关键任务道具一般有多种获取或提示途径，先看任务说明，别急着读档 |

探索是主线骨架；对话与规矩会在你走动时层层挂上。下一页：[对话与选择](./dialogue-choice)。
