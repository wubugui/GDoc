# 走进雾津

这页带你认清《寻狗记》讲的是个什么故事、雾津是个什么地方。读完你会知道：你扮演谁、大致做什么事、这游戏是个什么调调——然后就可以放心去点开下一页学操作了。

## 这是什么（30 秒看懂）

《寻狗记》是一个发生在**雾津**的探索、对话、解谜类单机游戏。民国年间，长江水雾常年罩着这座临江山城——本地人图省事，都管这儿叫**雾津**：码头湿、巷子窄、袍哥规矩多，城隍庙香火不断，义庄里却总有人半夜听见纸人沙沙响。

你操纵的角色是码头扒尸混混**关二狗**。他丢的不是什么宝贝，是条跟了他多年的土狗，名叫**二狗**。半文不白的游方道士**李天狗**凑上来帮忙，嘴上贫、手里却真会几手术。两个人在渡口、茶馆、义庄、土地庙之间穿街走巷，渐渐发现：寻狗这事，牵出的不只是狗链，还有人心、鬼神，以及雾津人代代口传的**规矩**。

打个比方：如果你熟悉的探索游戏是「地图上到处逛、跟人聊、时不时来一场紧要关头的抉择」，雾津就是把这套玩法泡进了长江水汽和民国市井里——多了口音、多了鬼神忌讳、也多了几分黑色幽默。

## 入门：新手怎么玩

### 你要做的事，一句话概括

**在雾津城里走动，跟人说话，守规矩、破困局，把二狗找回来。**

### 日常循环是这样的

```mermaid
flowchart LR
  A[在场景里走动] --> B[调查 / 对话 / 拾取]
  B --> C[接任务 · 记见闻]
  C --> D[遭遇抉择 · 小游戏]
  D --> E[推进主线与支线]
  E --> A
```

- **探索**：自由走动，调查可疑之处，进出不同街区。
- **对话**：跟 NPC 攀谈，选项会改变关系与后续剧情。
- **规矩**：学来的条文会在关键时刻给你多一条路，或拦住莽撞的选择。
- **险境**：叫魂、屏息、鬼打墙——有时得一口气憋到底。
- **档案**：人物簿、见闻录会随进度解锁，帮你拼出雾津全貌。

### 第一次上手，照这个顺序走

刚进游戏，别急着到处乱跑，按这个节奏摸清套路：

1. 先看一眼**任务面板**，知道当前目标是什么、该往哪走。
2. 在场景里走两步，留意屏幕上出没出**互动提示**——出现了就按互动键试试（调查、拾取、跟人搭话）。
3. 第一次跟 NPC 对话，把台词读完，别急着跳过——重庆话的市井腔头几句可能不顺耳，多读几句就顺了。
4. 遇到需要选选项的对话，选之前想想「关二狗会怎么答」，选完看看后续台词变了没有。
5. 走出这个场景前，翻翻**规矩本**和**档案**，看看有没有新解锁的条目。

这五步走完，你差不多就懂了雾津的基本节奏——后面每个场景都是这套动作的重复和加深。

## 进阶：玩深的技巧

### 雾津是什么地方

雾津不是一个笼统的「城」，它由几片风格很不一样的街区拼起来，你在故事里会依次撞见：

| 地方 | 你在故事里会遇见什么 |
|---|---|
| **渡口码头** | 纤夫、货郎、捞尸闲话；夜里有河灯、沉箱、水鬼传闻 |
| **城隍庙** | 叫魂、拜神、糖画摊；规矩与香火绑在一起 |
| **义庄** | 纸扎、丧仪、停灵；纸人不能乱碰 |
| **土地庙** | 小庙大用；李天狗常在这儿歇脚、念咒 |
| **阎王岭** | 城外险地；瘴气、鬼打墙，没准备好别硬闯 |

这几片地方不是孤立的关卡——同一个 NPC 可能这周在渡口卖货，剧情推进后挪到城隍庙庙会摆摊；同一条街，位面变了之后能走的路、能碰的人也会不一样（这个后面 [操作与界面](./controls) 会细讲）。

### 两个主角，一段配戏

- **关二狗**：码头混混出身，嘴硬心软，遇事先贫两句缓一缓，真到关头又豁得出去。
- **李天狗**：游方道士，穿着一身不太体面的道袍，嘴上没个正形，手底下的门道却是真的——规矩、符纸、念咒，他都懂几分。

两人一路的对话，是这游戏最大的乐趣来源之一——别把台词当纯粹的功能性提示跳过去，很多雾津的世界观细节和幽默感都藏在闲聊里。

### 对白的口音

对白多用**重庆话**（渝都口音），市井、泼辣、带点黑色幽默。第一次听不习惯，多走几处场景就顺耳了；实在卡壳，字幕会把意思兜住，不影响你判断该选哪个选项。

### 主题：规矩、鬼神、人心缠在一起

《寻狗记》不是纯解谜，也不是纯恐怖——它写的是**规矩、鬼神、人心**缠在一起时的雾津：有人怕鬼，有人借鬼吓人，有人靠规矩活命。关二狗嘴硬心软，李天狗似真似假；你做的每个选择，都会落在具体的人与具体街上，而不是一个抽象的好人/坏人分数。带着这个心态去玩对话和遭遇，会比单纯「选对答案」更有意思。

## 常见问题

**这游戏很恐怖吗？我怕鬼故事。**
有惊吓和险境元素（叫魂、鬼打墙一类），但不是纯恐怖游戏，更接近「志怪+市井」的调子。害怕突然的惊吓画面的话，可以先看看 [压力与险境](./pressure) 心里有个数。

**重庆话对白听不懂怎么办？**
正常，第一次接触多半会卡壳。字幕会把台词打全，读几段之后大部分俗语你会自己悟出意思；实在纠结某句话什么意思，不影响继续玩下去。

**这是恐怖解谜游戏还是纯剧情游戏？**
两者都沾一点：日常是探索+对话，紧要关头会切到一页多选的「遭遇」，偶尔还有长按蓄力的险境和几段小游戏。玩法细节见下面的阅读顺序。

**我需要先了解民国历史或袍哥文化背景吗？**
不需要。游戏会在对话和见闻录里边玩边给你讲，雾津的世界观是靠走动和聊天慢慢拼出来的，不用带任何前置知识进场。
