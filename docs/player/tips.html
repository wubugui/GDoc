提示与技巧

下面是不剧透主线结局的上手建议。你已经会走路、存档、对话的话，这些能少踩雾津的坑。

这是什么（30 秒看懂）​

这页不是新系统，是把规矩、压力与险境、物品、小游戏、档案 几页里最容易被忽略的实操经验，收拢成一份「老玩家会提醒新玩家」的清单。卡关、迷路、总在同一个长按翻车时，先回来翻这页,通常比硬着头皮乱试更快找到方向。

入门：开局前十分钟该做的事​

第一次进游戏，建议按这个顺序把习惯立起来：

  1. 按 Esc 打开暂停菜单，确认三个存档槽怎么用（见 存档与设置）。
  2. 按 Tab 打开任务面板，看清楚当前目标写的是什么。
  3. 打开一次背包（I）、规矩本（R）、档案（B）、地图（M），混个脸熟——不用看懂内容，先知道「在哪」。
  4. 跟第一个遇到的 NPC 把对话读完，别一路点跳过，重庆话口吻里常带任务关键词。
  5. 路过能调查的地方先按 E 调查——拾取和见闻往往在进对话前就摆在那儿等你。

进阶：各系统的老手习惯​

探索​

  • 土地庙 → 渡口 → 城隍庙 是早期高频三角，脚程熟了自己会抄近路。
  • 互动没提示：换站位、换时间（剧情推进后）、换位面后再来，很多热区是条件触发的。
  • 义庄、纸人相关：先调查再动手，规矩本和见闻会暗示能不能碰。
  • 阎王岭、城外夜路：带香烛符纸，存档再进（见 压力与险境）。

规矩与遭遇​

  • 遭遇里灰色选项不是 bug，是条件不够——翻规矩本、查背包，通常两者之一能解释。
  • 碎片漏收集，后面选项可能一直锁着；回头扫码头、庙侧、义庄角落，碎片来源见 规矩系统。
  • 「破规矩」有时开隐藏，但付代价；想稳就按象→理→术逐层学。

物品与经济​

  • 铜钱不富余：优先任务消耗品，糖画可试但别 all in。
  • 任务道具别当垃圾扔；说明里写「交给谁」的先留着。
  • 香烛、符纸在叫魂、遭遇里常要消耗，城隍庙铺子补货，多逛几家比价。

压力与险境​

  • 长按类互动一旦提示变成「别放手」一类措辞，往往是越过了不可反悔的关口——提前判断自己撑不撑得住，别临时反悔。
  • 鬼打墙绕回原地是常态，不是卡死，冷静按提示走，别乱按交互键。
  • 险境前的存档习惯比任何技巧都管用：开暂停菜单存一槽是正当玩法，不是作弊。

小游戏​

游戏技巧
糖画转盘可能要花钱；凶格问题不大，读档可再讨彩头
扎纸纸色、槽位对齐；「点睛」类收尾看清规矩本再答
水域限时关先存档；不同目标拉杆节奏不同，空竿别慌

档案与沉浸​

  • 每认识一个关键 NPC，翻人物簿看印象有没有更新。
  • 见闻录和任务交叉读，少漏「某地有调查」的提示。
  • 耳机开环境音：雨声、钟、河浪对氛围和险境节奏都有帮助。

心态​

《寻狗记》写的是寻狗牵出人鬼人心，不是纯收集或纯恐怖。关二狗嘴贱、李天狗可疑——人和规矩一样值得琢磨。卡关时按这个顺序排查：

  1. 对任务目标，看写的是去哪找谁。
  2. 扫规矩本与档案，找缺的是条件还是信息。
  3. 回早期场景看新热区，很多互动是条件触发才出现。
  4. 仍卡就读档换选项路线试试，雾津不少分支不能事后合并。

祝你在雾津湿鞋而行，早日把二狗找回家。

常见问题​

卡关了但任务面板没给新提示怎么办？ 先按上面「心态」的四步排查，绝大多数卡关是漏了某处调查或某条规矩碎片，而不是流程真的断了。

是不是每个选择都要读档试一遍才不亏？ 不需要。雾津的分支设计成「走出自己的一条路」，读档是给重大失误或想看另一条线的人准备的，不是强制的完美主义流程。

新手总在同一个长按翻车，是操作问题还是没学会规矩？ 两者都可能，优先检查对应规矩的术层学没学到位，再检查有没有该带的符纸香烛，最后才是手速和节奏问题。

探索时总觉得走空，是不是漏了什么？ 换个时间段（剧情推进后）、换个位面再回去看看，很多热区不是永久隐藏，而是条件还没到。

要不要每次经过 NPC 都搭话？ 建议搭，尤其是固定站位的角色，台词会随剧情更新，多聊不吃亏，还可能补上人物簿印象。
