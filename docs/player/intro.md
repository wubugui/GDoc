# 走进雾津

民国年间，长江水雾常年罩着**临江山城**。本地人管这儿叫**雾津**——码头湿、巷子窄、袍哥规矩多，城隍庙香火不断，义庄里却总有人半夜听见纸人沙沙响。

你扮演码头扒尸混混**关二狗**。他丢的不是什么宝贝，是条跟了他多年的土狗，名叫**二狗**。半文不白的游方道士**李天狗**凑上来帮忙，嘴上贫、手里却真会几手术。两个人在渡口、茶馆、义庄、土地庙之间穿街走巷，渐渐发现：寻狗这事，牵出的不只是狗链，还有人心、鬼神，以及雾津人代代口传的**规矩**。

## 雾津是什么地方

| 地方 | 你在故事里会遇见什么 |
|---|---|
| **渡口码头** | 纤夫、货郎、捞尸闲话；夜里有河灯、沉箱、水鬼传闻 |
| **城隍庙** | 叫魂、拜神、糖画摊；规矩与香火绑在一起 |
| **义庄** | 纸扎、丧仪、停灵；纸人不能乱碰 |
| **土地庙** | 小庙大用；李天狗常在这儿歇脚、念咒 |
| **阎王岭** | 城外险地；瘴气、鬼打墙，没准备好别硬闯 |

对白多用**重庆话**（渝都口音），市井、泼辣、带点黑色幽默。第一次听不习惯，多走几处场景就顺耳了。

## 你在游戏里做什么

一句话：**在雾津城里走动，跟人说话，守规矩、破困局，把二狗找回来。**

日常循环大致是这样：

```mermaid
flowchart LR
  A[在场景里走动] --> B[调查 / 对话 / 拾取]
  B --> C[接任务 · 记见闻]
  C --> D[遭遇抉择 · 小游戏]
  D --> E[推进主线与支线]
  E --> A
```

- **探索**：自由走动，调查可疑之处，进出不同街区。
- **对话**：跟 NPC 攀谈，选项会改变关系与后续剧情。
- **规矩**：学来的条文会在关键时刻给你多一条路，或拦住莽撞的选择。
- **险境**：叫魂、屏息、鬼打墙——有时得一口气憋到底。
- **档案**：人物簿、见闻录会随进度解锁，帮你拼出雾津全貌。

## 氛围与主题

《寻狗记》不是纯解谜，也不是纯恐怖——它写的是**规矩、鬼神、人心**缠在一起时的雾津：有人怕鬼，有人借鬼吓人，有人靠规矩活命。关二狗嘴硬心软，李天狗似真似假；你做的每个选择，都会落在具体的人与具体街上。

准备好湿鞋进雾津，就从 [操作与界面](./controls) 开始。
