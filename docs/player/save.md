# 存档与设置

雾津的故事线岔路多，建议在关键抉择前、进险境前**存个档**。游戏提供快速存读，不用退回主菜单。

---

## 快速存档与读档

| 按键 | 作用 |
|---|---|
| `F5` | **快速存档**——立刻写入当前进度 |
| `F6` | **快速读取**——回到上一次快速存档的状态 |

快速档通常只有**一个槽位**：再按一次 `F5` 会覆盖之前的快速档。想保留「试错前」的状态，请在重大选择前先存。

:::warning[读档会回滚]
读档后，自该存档点之后的探索、对话选项、物品变动都会丢失。叫魂、鬼打墙、商店购物都一样——读档即重来。
:::

---

## 存档里记了什么

一次完整存档会保存当前游玩状态，包括但不限于：

| 类别 | 内容 |
|---|---|
| **位置** | 你在哪个场景、站在哪儿 |
| **任务** | 主线、支线进度与完成条件 |
| **物品** | 背包里有什么、数量多少 |
| **规矩** | 已掌握条文、碎片收集情况 |
| **旗标** | 剧情里各种「发生过没有」的记录（如跟谁说过话、门是否打开） |
| **叙事** | 故事推进到哪一阶段、当前位面等 |
| **档案** | 已解锁的人物簿、见闻录条目 |

因此读档不仅能「回到打架前」，也能纠正选错选项、花错钱、在险境里失手的情况。

---

## 什么时候建议存档

| 时机 | 原因 |
|---|---|
| 进**城隍庙**叫魂、过场长按前 | 失败可能要重来一整段 |
| **遭遇**多选项、耗物品前 | 选项往往不可撤销 |
| 进**阎王岭**或鬼打墙相关区域前 | 险境惩罚重 |
| **商店**大笔花钱前 | 读档可反悔（若你接受用读档试错） |
| 小游戏（水域限时捞物）开始前 | 限时关失手成本高 |

不剧透具体剧情节点；养成「进陌生区域先 F5」的习惯，在雾津很管用。

---

## 设置项

游戏内设置以你使用的版本（浏览器版或独立导出包）为准，常见包括：

| 设置 | 说明 |
|---|---|
| **音量** | 背景音乐、环境音、音效分开或总音量 |
| **文字速度** | 对话逐字显示的快慢 |
| **窗口 / 全屏** | 独立包常见；浏览器版随浏览器窗口 |

设置一般**随存档或本地配置保存**，与剧情进度分开。调音量不会改任务状态。

---

## 与探索、险境的关系

- 探索时随时可快速存档（除非剧情暂时锁定输入）。
- [压力与险境](./pressure) 里长按失败、被打断，读档是合法策略——游戏没设计成必须一命通关。
- 多周目或新游戏+：若版本提供「新游戏」入口，会从开局配置重来；与 `F6` 读快速档不是一回事。
