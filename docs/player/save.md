# 存档与设置

这页讲清楚：怎么存档读档、一次存档到底记了哪些东西、以及什么时候必须存一下。读完你就知道怎么在雾津放心地「试错」——选错话、扛不住险境，都有退路。

---

## 这是什么（30 秒看懂）

雾津的故事线岔路多，一旦选下去，很多分支不能回头合并。为此游戏提供了**快速存读档**：不用退回主菜单，一个按键就能记住此刻状态，一个按键就能回到上次记住的状态。你可以把它理解成雾津给你的一份「悔棋权」——用不用、什么时候用，你自己拿主意。

---

## 入门：新手怎么玩

### 第一次存读档，照这几步做

1. 走到一处安全的场景（不在对话/遭遇/小游戏中），按 `F5`。屏幕通常会有个简短的提示或音效，代表存档成功。
2. 随便走两步、换个位置。
3. 按 `F6` 读档，确认角色回到了刚才存档时的位置和状态。
4. 记住这个组合键：**进陌生场景、险境、大额消费前先 `F5`**，这个习惯贯穿整本手册后面所有页面。

### 快速存档与读档

| 按键 | 作用 |
|---|---|
| `F5` | **快速存档**——立刻写入当前进度 |
| `F6` | **快速读取**——回到上一次快速存档的状态 |

快速档通常只有**一个槽位**：再按一次 `F5` 会覆盖之前的快速档。想保留「试错前」的状态，请在重大选择前先存。

:::warning[读档会回滚]
读档后，自该存档点之后的探索、对话选项、物品变动都会丢失。叫魂、鬼打墙、商店购物都一样——读档即重来。
:::

## 进阶：玩深的技巧

### 存档里记了什么

一次完整存档会保存当前游玩状态，包括但不限于：

| 类别 | 内容 |
|---|---|
| **位置** | 你在哪个场景、站在哪儿 |
| **任务** | 主线、支线进度与完成条件 |
| **物品** | 背包里有什么、数量多少 |
| **规矩** | 已掌握条文、碎片收集情况 |
| **旗标** | 剧情里各种「发生过没有」的记录（如跟谁说过话、门是否打开） |
| **叙事** | 故事推进到哪一阶段、当前位面等 |
| **档案** | 已解锁的人物簿、见闻录条目 |

因此读档不仅能「回到打架前」，也能纠正选错选项、花错钱、在险境里失手的情况——它是一个完整的时间点快照，不是只回血、只回位置。

### 什么时候建议存档

| 时机 | 原因 |
|---|---|
| 进**城隍庙**叫魂、过场长按前 | 失败可能要重来一整段 |
| **遭遇**多选项、耗物品前 | 选项往往不可撤销 |
| 进**阎王岭**或鬼打墙相关区域前 | 险境惩罚重 |
| **商店**大笔花钱前 | 读档可反悔（若你接受用读档试错） |
| 小游戏（水域限时捞物）开始前 | 限时关失手成本高 |
| 一段对话出现多个不确定的选项时 | 想先看看某个选项的后果，可以存档试探，再读档走另一条 |

不剧透具体剧情节点；养成「进陌生区域先 `F5`」的习惯，在雾津很管用。

### 设置项

游戏内设置以你使用的版本（浏览器版或独立导出包）为准，常见包括：

| 设置 | 说明 |
|---|---|
| **音量** | 背景音乐、环境音、音效分开或总音量 |
| **文字速度** | 对话逐字显示的快慢 |
| **窗口 / 全屏** | 独立包常见；浏览器版随浏览器窗口 |

设置一般**随存档或本地配置保存**，与剧情进度分开。调音量不会改任务状态；调文字速度也不影响任何判定，纯是照顾阅读习惯——习惯快速读文字的玩家可以调快一点，省得追字幕耽误看画面。

### 和探索、险境的关系

- 探索时随时可快速存档（除非剧情暂时锁定输入）。
- [压力与险境](./pressure) 里长按失败、被打断，读档是合法策略——游戏没设计成必须一命通关。
- 多周目或新游戏+：若版本提供「新游戏」入口，会从开局配置重来；与 `F6` 读快速档不是一回事，误触了不会覆盖你的进度存档。

## 常见问题

**快速档只有一个吗？能存好几个不同进度吗？**
快速存读档默认只占**一个槽位**，覆盖式保存。如果想同时保留「试错前」和「试错后」两个版本，建议先手动过一遍关键抉择再考虑要不要读档回退，而不是指望多槽位并存。

**读档会不会影响糖画、水域这类小游戏的结果？**
会。小游戏的过关结果也算在存档快照里，读档等于把小游戏也退回读档前的状态，可以重新挑战。

**没存档就退出游戏，进度还在吗？**
不会自动保存，退出前没手动存档的部分都会丢。稳妥的做法是**每次告一段落都手动 `F5` 一次**，别依赖「关游戏会自动帮我存」的假设。

**读档能不能跳过已经看过的对话，直接到选项那一句？**
读档会精确回到存档那一刻，之后的台词、选项都要重新走一遍——它不是「剧情快进」工具，是「回到某个时间点」工具。

**能不能云存档，换设备继续玩？**
能不能云同步取决于你所在的发行渠道有没有提供这项功能；本页说明的是游戏内**快速存读档**机制本身，不涉及具体平台的云同步功能。
