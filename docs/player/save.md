# 存档与设置

这页讲清楚：怎么存档读档、一次存档到底记了哪些东西、以及什么时候必须存一下。读完你就知道怎么在雾津放心地「试错」——选错话、扛不住险境，都有退路。

---

## 这是什么（30 秒看懂）

雾津的故事线岔路多，一旦选下去，很多分支不能回头合并。为此游戏提供了**暂停菜单里的三个存档槽**：在探索状态按 `Esc`，选一个槽写入当前进度，需要时再从同一菜单读回。你可以把它理解成雾津给你的三份「悔棋权」——用不用、什么时候用，你自己拿主意。

:::note
没有独立的「快速存档热键」。编辑器里的 `F5` 是**运行预览**，只给改内容的人用，玩家游玩路径请走暂停菜单。
:::

## 入门：新手怎么玩

### 第一次存读档，照这几步做

1. 走到一处安全的场景（不在对话 / 遭遇 / 过场 / 小游戏中），按 `Esc` 打开暂停菜单。
2. 选一个空槽或可覆盖的槽，执行**保存**。界面会提示成功。
3. 随便走两步、换个位置。
4. 再按 `Esc`，从刚才那个槽**读取**，确认角色回到了存档时的位置和状态。
5. 养成习惯：**进陌生场景、险境、大额消费、重大选项前先存一槽**。

### 三槽存读档

| 项 | 说明 |
|---|---|
| **入口** | 探索状态按 `Esc` → 暂停菜单 |
| **槽位** | 三个独立槽，可分别保存不同进度 |
| **覆盖** | 对同一槽再次保存会覆盖该槽旧档 |
| **不安全状态** | 对话、遭遇、过场、小游戏进行中通常**拒绝存档**——先结束或退出到探索再存 |

:::warning[读档会回滚]
读档后，自该存档点之后的探索、对话选项、物品变动都会丢失。叫魂、鬼打墙、商店购物都一样——读档即重来。
:::

## 进阶：玩深的技巧

### 存档里记了什么

一次完整存档会保存当前游玩状态，包括但不限于：

| 类别 | 内容 |
|---|---|
| **位置** | 你在哪个场景、站在哪儿 |
| **任务** | 主线、支线、活计进度与完成情况 |
| **物品** | 背包里有什么、数量多少 |
| **规矩** | 已掌握条文、碎片收集情况 |
| **旗标** | 剧情里各种「发生过没有」的记录 |
| **叙事** | 故事推进到哪一阶段、当前位面等 |
| **档案** | 已解锁的人物簿、见闻录条目 |

因此读档不仅能「回到打架前」，也能纠正选错选项、花错钱、在险境里失手——它是完整时间点快照。

### 什么时候建议存档

| 时机 | 原因 |
|---|---|
| 进**城隍庙**叫魂、过场长按前 | 失败可能要重来一整段 |
| **遭遇**多选项、耗物品前 | 选项往往不可撤销 |
| 进**阎王岭**或鬼打墙相关区域前 | 险境惩罚重 |
| **商店**大笔花钱前 | 读档可反悔（若你接受用读档试错） |
| 小游戏（水域限时捞物）开始前 | 限时关失手成本高 |
| 一段对话出现多个不确定的选项时 | 想先看看某个选项的后果，可以存档试探 |

不剧透具体剧情节点；养成「进陌生区域先开菜单存一槽」的习惯，在雾津很管用。

### 设置项

| 设置 | 说明 |
|---|---|
| **音量** | 背景音乐、环境音、音效分开或总音量 |
| **文字速度** | 对话逐字显示的快慢 |
| **窗口 / 全屏** | 独立包常见；浏览器版随浏览器窗口 |

设置一般**随存档或本地配置保存**，与剧情进度分开。

### 和探索、险境的关系

- 探索时随时可开暂停菜单存档（除非剧情暂时锁定输入）。
- [压力与险境](./pressure) 里长按失败、被打断，读档是合法策略——游戏没设计成必须一命通关。
- 多周目或新游戏：若版本提供「新游戏」入口，会从开局配置重来；与读某一槽不是一回事。

## 常见问题

**三个槽怎么分配比较稳？**
常见分法：一槽「稳进度」、一槽「试错前」、一槽「险境前」。别三个槽都盖成同一次进度。

**读档会不会影响糖画、水域这类小游戏的结果？**
会。小游戏的过关结果也算在存档快照里，读档等于把小游戏也退回读档前的状态。

**没存档就退出游戏，进度还在吗？**
不会自动保存，退出前没手动存档的部分都会丢。稳妥的做法是每次告一段落都开菜单存一次。

**为什么对话里按了存档没反应？**
对话、遭遇、过场、小游戏中属于不安全状态，系统会拒绝写入。先结束当前段落回到探索，再 `Esc` 保存。

**能不能云存档，换设备继续玩？**
取决于发行渠道有没有提供；本页说明的是游戏内三槽存读档机制本身。
