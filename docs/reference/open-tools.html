工具打开方式

只回答：我想用某件工具，该怎么打开？ 在游戏仓库根目录执行下列命令，或点对应按钮。

更完整的用途说明见 工具速查表；概念说明见 编辑器 · 工具打开方式。

工具命令

Web 控制台（工具箱）./dev.sh console

常见问题​

主编辑器和 Web 控制台要不要同时开着？ 可以，也建议这样：主编辑器负责改内容，控制台负责一键跑游戏/构建/测试，两边各司其职，互不冲突。

独立窗口工具（图对话、视差编辑器等）和主编辑器改的是不是一回事？ 是同一份工程内容，只是换了个专门界面编辑。改完记得在那个工具自己的窗口里保存，别以为主编辑器的 Ctrl+S 会连带存掉独立窗口里的改动。

命令名记不住怎么办？ 不用背——Web 控制台里点按钮效果和敲命令完全一样，日常优先点按钮；只有远程/无图形界面环境才需要背命令。

同一个工具重复打开会怎样？ 一般会占用同一个端口/窗口，重复开容易报错或互相打架。先看看是不是已经开着了，没有再开新的。
