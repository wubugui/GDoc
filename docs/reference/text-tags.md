# 富文本标签速查

玩家看到的对白、描述、档案正文里，用 **`[类型:标识]`** 插入动态引用——运行时自动换成当前名字、数值或链接，改登记名全文跟着变。

入门见 **[怎么写带引用的文本](../editors/concepts/rich-text)**。

---

## 八种引用标签

| 标签写法 | 引用什么 | 典型句子 |
|---|---|---|
| `[string:…]` | 文本库里的固定条目 | 系统统一称呼、格式化用语 |
| `[flag:…]` | 旗标当前值（可当文字显示） | 随进度变的称谓、计数 |
| `[item:…]` | 物品显示名 | 「你摸了摸 [item:狗哨]」 |
| `[npc:…]` | 角色名（可加 `@场景实例` 消歧） | 「[npc:关二狗] 哼了一声」 |
| `[player:…]` | 玩家化身名、称谓等 | 「[player:name]，回来啦？」 |
| `[quest:…]` | 任务标题 | 「关于 [quest:寻狗记] 的线索」 |
| `[rule:…]` | 规矩条文摘要 | 「违反 [rule:城规三则] 者…」 |
| `[scene:…]` | 场景显示名 | 「前往 [scene:雾津码头]」 |

**怎么插：** 富文本框边 **插入引用** 按钮 → 选类型 → 选条目 → 自动生成，比手打不易错。

**手打注意：** 标识必须与游戏里登记一致；错了可能显示空白，编辑器不一定当场报错。

## 插图标签 `[img:短名]`

| 项 | 说明 |
|---|---|
| **干什么** | 正文内嵌一张已登记图片 |
| **短名从哪来** | [叠图](../editors/panels/overlay) 面板登记的短名 |
| **哪里能点按钮插** | **仅 [档案](../editors/panels/archive)** 正文 |
| **别处** | 手打标签运行时可能显示，但编辑器**不校验、不引导**——属于盲区行为，保存还可能被抹（见 [危险区](./danger-zone)） |

雾津见闻录配图、书籍插图：在档案里用按钮插；别在场景热区手写赌保存。

---

## 哪些字段不能插标签

以下多为**纯标识**或内部引用，界面是单行文本，**不可**插引用标签：

- 各种内部 id 字段（任务 id、场景 id、物品 id 等）
- 规矩碎片上只读的规矩引用行

对白、描述、任务说明、规矩正文、文本库字符串值——走富文本框的**可以**插。

## 常见出现位置

| 面板 | 字段 |
|---|---|
| [图对话](../editors/panels/dialogue-graph) | 台词、选项文案 |

| [档案](../editors/panels/archive) | 条目正文（可插图） |
| [任务](../editors/panels/quest) | 标题、说明 |
| [规矩](../editors/panels/rule) | 各层正文 |
| [遭遇](../editors/panels/encounter) | 选项与结果文字 |

| [场景](../editors/panels/scene) | 热区显示标签 |

## 快速示例

**对白：**

> [npc:关二狗]：这 [item:狗哨] 不是你的吧，[player:name]？

**档案（带图）：**

> 雾津渡口旧照如下：[img:wuji_dock_sketch]

操作：图对话里点 **插入 → 物品 / NPC / 玩家**；档案里另可 **插入 → 插图**。
