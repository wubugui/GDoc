动作大全

主编辑器各面板的「动作列表」共用同一套类型。下表收录策划最常用的约五十种；完整只读目录在 注册与扩展 → 动作总表 面板。

编排入门见 怎么编排动作。动作里不能塞条件——条件放外层条件槽。

组合与分支​

动作（策划说法）干什么什么时候用
顺序执行一组从上到下依次跑子动作一件事要连做好几步：给物→改旗标→播提示
让玩家择一弹出选项，每条选项挂子动作遭遇式分支、可取消的二次确认
随机走一支按概率或随机挑上下两支之一彩蛋、随机台词、抽奖感演出
延时再执行指定游戏日后跑一串动作隔几天邮差送信、定时解锁
按规矩槽择一在规矩提供的选项里选，各槽挂结果规矩判定后让玩家选「认账 / 狡辩 / 装傻」

对话与叙事​

动作干什么什么时候用
开始图对话打开指定对白图与入口NPC 交互、热区观察转对白
播脚本对白不经图对话，直接弹一句过场感旁白、系统播报
发出叙事信号通知叙事状态机「发生了某事」任务线推进、位面切换的正规入口
启动剧本激活一条剧本线多线叙事开局
推进剧本阶段把剧本推到某阶段某状态章节节拍、结局分支
完成剧本标记剧本线完结章末收束
揭示文档解锁档案里一份模糊文档调查玩法、凑齐线索后显形
加入档案条目人物簿/见闻录等新增可读页第一次见到某人、获得传闻
等待点击继续暂停到玩家点一下分段演出、让玩家读完再往下
等待若干毫秒纯时间延迟字幕节奏、动作之间留白
提示

位面切换优先在 叙事状态机 里用「激活位面」状态字段；动作里的「激活位面」「关闭位面」是逃生舱式快捷，别与状态机双写冲突。

旗标与计数​

动作干什么什么时候用
设置旗标把某个旗标设为指定值进度开关、剧情分支记忆
追加旗标文本往可追加型旗标后拼接一段日记式累加记录
旗标加减数值型旗标 ±N好感、计数、资源累计

任务​

动作干什么什么时候用
更新任务推进、完成或重置某任务对白结束交差、拾取触发任务更新

物品与背包​

动作干什么什么时候用
给予物品背包增加指定物品奖励、拾取热区、商店外赠送
移除物品扣减背包物品交付任务、消耗道具
给予钱币加钱赏金、拾取铜钱
扣钱币减钱罚款、购物前置扣款
执行拾取按名称捡起（可绑物品 id）动作链里模拟捡地上东西
商店购买按价购买并扣钱脚本化购物、无需开商店 UI
丢弃背包物品从背包扔掉剧情没收、献祭道具

规矩​

动作干什么什么时候用
给予整条规矩玩家获得规矩条目教学关发「城规三则」
解锁规矩层规矩某一层变为可见/可验证调查推进、验证成功
给予规矩碎片发一条碎片线索拼出完整规矩前的小纸条
启用规矩选项打开某规矩的交互选项槽遭遇前挂可选辩词
关闭规矩选项关掉选项槽阶段结束、防止重复选

场景与实体​

动作干什么什么时候用
切换场景切到另一张图与出生点进门、坐船、过场落点
换场景（带镜头）切场景并可带镜头坐标需要指定落点镜头时
移动实体到坐标把 NPC/热区等挪到某点剧情后人物离场、物件复位
面向某方向让实体转向对峙、转身离去
设实体字段改场景里某实体的一个属性改显示名、开关等检视器级字段
设热区展示图换热区头上小图状态变化（门开/关、灯亮）
临时改热区朝向仅本次改展示朝向短演出，不写入存档
开关实体当场显示/隐藏实体路人突然出现
持久隐藏实体写入存档的显隐剧情后某人永久不在
持久改 NPC 位置存档里记住 NPC 站哪任务后 NPC 搬家
持久改 NPC 动画存档里记住站姿受伤后一直瘸着
停 NPC 巡逻当场停走路径触发对白时站定
持久停/启巡逻存档级停或恢复巡逻任务完成后不再闲逛
开关区域当场启用/禁用区域触发临时封路
持久开关区域存档级封路或开放炸桥后永久不通
持久开关热区存档级热区可用性捡过的道具不再显示
设深度地板偏移调本场景走路踩层景深玩法临时抬升
恢复深度地板还原默认踩层演出结束复位

镜头与画面​

动作干什么什么时候用
设镜头缩放推拉镜头倍率强调表情、俯瞰街巷
恢复场景默认镜头还原进场景时的镜头演出后还给玩家操控感
渐变缩放镜头带动画地推拉柔和运镜
渐变恢复镜头带动画地还原过场感收束
世界淡出黑画面渐黑转场、日终
世界从黑淡入画面渐亮醒来、进新场景
显示叠图屏幕上叠一张登记图插图、线索特写
隐藏叠图去掉叠图特写结束
叠图渐变切换两张叠图过渡照片慢慢变清晰

音频​

动作干什么什么时候用
播放背景音乐切换 BGM进庙、战斗、温馨室内
停止背景音乐停 BGM静音过场、突出音效
播放音效播一次短音脚步、锣、开门
停环境音停场景环境循环进室内关雨声

表现与 UI​

动作干什么什么时候用
显示系统提示屏幕角弹出短消息获得物品、任务更新
显示表情气泡实体头顶表情惊讶、省略号
显示说话气泡实体头顶短句不用开整段对白
显示表情并等待表情播完才继续时间轴严格同步
显示说话并等待气泡播完才继续连续节拍对白
播 NPC 动画切角色动画状态挥手、下蹲、战斗
播放信号 Cue触发预设表现包复杂演出一次打包
开始过场播放时间轴过场大演出、不可操控段落

遭遇、商店、小游戏​

动作干什么什么时候用
开始遭遇进入选项式遭遇盘问、议价、危机抉择
打开商店弹出商店界面香烛铺、当铺
开始水域小游戏拉起捞物玩法河边摸鱼、捞鞋
开始转盘小游戏拉起糖画转盘街头赌博感玩法
开始扎纸小游戏拉起扎纸订单丧葬线手工玩法
开始临场长按拉起蓄力条 QTE推门、稳烛火

玩家状态​

动作干什么什么时候用
设玩家化身换玩家外观动画包换装、变装
恢复默认化身还原初始外观变装结束
设生命值直接设血量剧情定伤
增加生命值加血吃药、休息
减少生命值扣血陷阱、战斗（新内容优先用加减血，旧扣血类已下架）
重置生命值回满或回默认安全屋、章节重置
触发死亡系绳走死亡系绳逻辑特殊死亡演出（非普通扣血）
设气味赋予/加强某气味嗅觉玩法
清除气味去掉气味洗澡、换场景
嗅闻执行一次嗅闻判定找物、追踪

日程​

动作干什么什么时候用
结束当日推进游戏日、触发日终流程睡觉、坐船过夜

调试专用（正式内容勿用）​

动作干什么什么时候用
硬设叙事状态直接跳到某叙事状态仅开发调试
弹窗打印动作参数开发时看参数仅排错
