# 动作大全

主编辑器各面板的「动作列表」共用同一套类型，全库共一百来种。下表已按用途分类**尽量列全**你在各面板下拉里能挂上的动作；如果下拉里还有一条没找到对应说法，去 **注册与扩展 → 动作总表** 面板核对官方名字（那里是只读目录，不能编排）。

编排入门见 **[怎么编排动作](../editors/concepts/actions)**。动作里**不能**塞条件——条件放外层条件槽。

---

## 组合与分支

| 动作（策划说法） | 干什么 | 什么时候用 |
|---|---|---|
| **顺序执行一组** | 从上到下依次跑子动作 | 一件事要连做好几步：给物→改旗标→播提示 |
| **让玩家择一** | 弹出选项，每条选项挂子动作 | 遭遇式分支、可取消的二次确认 |
| **随机走一支** | 按概率或随机挑上下两支之一 | 彩蛋、随机台词、抽奖感演出 |
| **延时再执行** | 指定游戏日后跑一串动作 | 隔几天邮差送信、定时解锁 |
| **按规矩槽择一** | 在规矩提供的选项里选，各槽挂结果 | 规矩判定后让玩家选「认账 / 狡辩 / 装傻」 |

## 对话与叙事

| 动作 | 干什么 | 什么时候用 |
|---|---|---|
| **开始图对话** | 打开指定对白图与入口 | NPC 交互、热区观察转对白 |
| **播脚本对白** | 不经图对话，直接弹一句 | 过场感旁白、系统播报 |
| **发出叙事信号** | 通知叙事状态机「发生了某事」 | 任务线推进、位面切换的正规入口 |
| **激活位面** | 把玩家切到某个位面层 | 逃生舱式快捷切位面，正规入口优先走叙事状态机 |
| **关闭位面** | 把某个位面层关闭 | 同上，收场时配合用 |
| **启动剧本** | 激活一条剧本线 | 多线叙事开局 |
| **推进剧本阶段** | 把剧本推到某阶段某状态 | 章节节拍、结局分支 |
| **完成剧本** | 标记剧本线完结 | 章末收束 |
| **揭示文档** | 解锁档案里一份模糊文档 | 调查玩法、凑齐线索后显形 |
| **加入档案条目** | 人物簿/见闻录等新增可读页 | 第一次见到某人、获得传闻 |
| **等待点击继续** | 暂停到玩家点一下 | 分段演出、让玩家读完再往下 |
| **等待若干毫秒** | 纯时间延迟 | 字幕节奏、动作之间留白 |

:::tip
位面切换优先在 **叙事状态机** 里用「激活位面」状态字段；动作里的「激活位面」「关闭位面」是逃生舱式快捷，别与状态机双写冲突。
:::

## 旗标与计数

| 动作 | 干什么 | 什么时候用 |
|---|---|---|

## 物品与背包

| 动作 | 干什么 | 什么时候用 |
|---|---|---|
| **给予物品** | 背包增加指定物品 | 奖励、拾取热区、商店外赠送 |
| **移除物品** | 扣减背包物品 | 交付任务、消耗道具 |
| **给予钱币** | 加钱 | 赏金、拾取铜钱 |
| **扣钱币** | 减钱 | 罚款、购物前置扣款 |
| **执行拾取** | 按名称捡起（可绑物品 id） | 动作链里模拟捡地上东西 |
| **商店购买** | 按价购买并扣钱 | 脚本化购物、无需开商店 UI |
| **丢弃背包物品** | 从背包扔掉 | 剧情没收、献祭道具 |

## 规矩

| 动作 | 干什么 | 什么时候用 |
|---|---|---|
| **给予整条规矩** | 玩家获得规矩条目 | 教学关发「城规三则」 |
| **解锁规矩层** | 规矩某一层变为可见/可验证 | 调查推进、验证成功 |

| **关闭规矩选项** | 关掉选项槽 | 阶段结束、防止重复选 |

## 场景与实体

| 动作 | 干什么 | 什么时候用 |
|---|---|---|
| **切换场景** | 切到另一张图与出生点 | 进门、坐船、过场落点 |
| **换场景（带镜头）** | 切场景并可带镜头坐标 | 需要指定落点镜头时 |
| **移动实体到坐标** | 把 NPC/热区等挪到某点 | 剧情后人物离场、物件复位 |
| **直接设置实体坐标** | 与上面同类的另一种挪动动作，一步落到目标点 | 需要瞬间归位而非慢慢移过去时优先试这个 |
| **面向某方向** | 让实体转向 | 对峙、转身离去 |
| **设实体字段** | 改场景里某实体的一个属性 | 改显示名、开关等检视器级字段 |
| **设热区展示图** | 换热区头上小图 | 状态变化（门开/关、灯亮） |
| **临时改热区朝向** | 仅本次改展示朝向 | 短演出，不写入存档 |
| **开关实体** | 当场显示/隐藏实体 | 路人突然出现 |
| **持久隐藏实体** | 写入存档的显隐 | 剧情后某人永久不在 |
| **持久改 NPC 位置** | 存档里记住 NPC 站哪 | 任务后 NPC 搬家 |
| **持久改 NPC 动画** | 存档里记住站姿 | 受伤后一直瘸着 |
| **播动画并持久记住** | 播一次动画，同时把这个状态写进存档 | 演一次受伤动作、之后一直保持这个姿势 |
| **停 NPC 巡逻** | 当场停走路径 | 触发对白时站定 |
| **持久停/启巡逻** | 存档级停或恢复巡逻 | 任务完成后不再闲逛 |
| **开关区域** | 当场启用/禁用区域触发 | 临时封路 |
| **持久开关区域** | 存档级封路或开放 | 炸桥后永久不通 |
| **持久开关热区** | 存档级热区可用性 | 捡过的道具不再显示 |
| **设深度地板偏移** | 调本场景走路踩层 | 景深玩法临时抬升 |
| **恢复深度地板** | 还原默认踩层 | 演出结束复位 |

## 过场专属（仅时间轴逻辑步可用）

过场时间轴里的「逻辑步」只能挂一份**更小的动作白名单**（约三十余种，覆盖上面大部分表现类与旗标类动作），下面两个是**只在这里才有**、别处面板挂不了的：

| 动作 | 干什么 | 什么时候用 |
|---|---|---|
| **过场生成临时演员** | 在过场画面上临时冒出一个演员 | 需要一个只在这场过场里露脸、平时场景里没有的配角 |
| **过场移除临时演员** | 把上面生成的临时演员撤走 | 演出收尾、恢复画面干净 |

编排前先在过场编辑器的动作下拉里确认能不能选到——挂不进去说明不在这份白名单里，见 [过场](../editors/panels/cutscene) 与 [危险区](./danger-zone)。

## 转盘小游戏专属回调

以下四个只用在**转盘小游戏**的分格/开场回调里，别的面板选不到：

| 动作 | 干什么 | 什么时候用 |
|---|---|---|
| **转盘显示台词** | 分格触发时弹一句台词气泡 | 转到某格时角色吐槽一句 |
| **收起当前台词** | 把刚弹出的那句收起来 | 台词念完手动收场 |
| **收起全部台词** | 一次性清掉转盘上所有台词气泡 | 切换到下一轮前清屏 |
| **重置转盘指针** | 指针复位回初始角度 | 一局结束、再来一次前复位 |

## 玩家状态

| 动作 | 干什么 | 什么时候用 |
|---|---|---|
| **设玩家化身** | 换玩家外观动画包 | 换装、变装 |
| **恢复默认化身** | 还原初始外观 | 变装结束 |
| **设生命值** | 直接设血量 | 剧情定伤 |
| **增加生命值** | 加血 | 吃药、休息 |
| **减少生命值** | 扣血 | 陷阱、战斗（新内容优先用加减血，旧扣血类已下架） |
| **重置生命值** | 回满或回默认 | 安全屋、章节重置 |
| **触发死亡系绳** | 走死亡系绳逻辑 | 特殊死亡演出（非普通扣血） |
| **设气味** | 赋予/加强某气味 | 嗅觉玩法 |
| **清除气味** | 去掉气味 | 洗澡、换场景 |
| **嗅闻** | 执行一次嗅闻判定 | 找物、追踪 |
