# 工具速查表

参考区不重复维护整张表。完整 **GameDraft 全部编辑器/工具** 一览（按域分组、用途、形态、启动方式、文档状态）在：

**[编辑器 · 工具速查表](../editors/tool-matrix)**

该页由单一数据源渲染，加删工具时数字与表格自动同步。

## 按域粗分（记大类用）

| 域 | 典型工具 |
|---|---|
| 主编辑与生产 | 主编辑器、生产工作台、Web 控制台 |
| 资产 | 资源浏览、入库、图片缩放、视频转图集 |
| 图与叙事 | 图对话、通用图、叙事状态机 |
| 渲染 | 场景深度、视差、滤镜、光照体积 |
| 文案 | 文案管理 |
| 模拟 | 编年史模拟 |
| 语言服务 | JSON 语言服务 |
| 治理 | Skill/Workflow 治理台 |
