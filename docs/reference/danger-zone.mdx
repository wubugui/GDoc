# 危险区：真正要当心的与其实没事的

先说结论，别被吓着：**编辑器其实相当稳。** 你在面板里看得见、改得动的东西，存了就在；**连它自己不认识的字段，多数地方也会原样保留**（一次"去所有权化"改造的成果）。

真正需要你知道的，只有下面**三类**，外加一张"其实没事"的宽心表。

:::tip[一句话准则]
面板里**有的**，放心改；面板里**没有**的那几个活字段（见下方「盲区」），换专项工具或手改；其余"保存时被清掉"的，都是**废弃的旧字段**，清掉不影响游戏。
:::

---

## 一、盲区：有些活数据，主面板改不到（得换工具）

这些字段游戏运行时**真的会用**，但主编辑器对应面板**没有入口**。注意：它们**不会被丢**（编辑器原样保留），只是你在这儿**编辑不了**，得换地方。

| 够不着的东西 | 去哪改 |
|---|---|
| 场景**多层背景**、深度/遮挡碰撞主体 | **[场景深度工具](../editors/render-domain/scene-depth-editor)** |
| **玩家化身**外观细节 | **[玩家化身面板](../editors/panels/avatar)** |
| 旗标的**「迁移」「运行时」块** | 暂无界面，手改或找维护者 |
| 实体像素密度匹配 | 暂无界面，手改 |

:::warning[一个真隐患]
场景面板的**「重新导入背景」按钮**：如果一张场景本来有多层背景，用这个按钮换图会把它**压成单层**。多层背景请一律走 [场景深度工具](../editors/render-domain/scene-depth-editor)，别用主面板的重导。
:::

---

## 二、显式的破坏性操作（你主动点才会发生）

这些不是"悄悄丢"，是你**明确做了某个动作**才触发的后果——知道了就不会误伤。

- **把区域从「标准」改成「深度地板」** → 会清掉它的进入 / 停留 / 离开动作（还有气味）。深度地板区本来就不触发这些，所以正常用没问题；但**切过去再切回来，原来的动作就找不回了**。
- **滤镜 id = 文件名，位面「现世」的 id** → 只读，改名等于删掉重建。
- **删小游戏实例** → 登记表里没了，但**磁盘上那个旧实例文件不会自动删**，需要你手动清。
- **水域小游戏**的显示尺寸 / 命中半径**留空** → 表示"用品类默认"，保存时不写这两个键（不是丢，是刻意省略）。

## 三、自动清理的废弃字段（不是丢你的东西）

保存时编辑器会顺手删掉一些**旧格式 / 已废弃**的字段。这些**运行时早就不用了**（或有兜底、但当前内容一处没在用），清掉不影响游戏——是好事（保持数据干净），不是危险。

- NPC 的旧对白引用、区域旧矩形坐标与规矩槽位
- 任务的旧式「单一下一任务 id」（当前所有任务都已用新的连线格式）
- 过场顶层的旧「命令列表」（现在用的是「步骤」）
- 规矩的几个旧字段；转盘一个已废弃的「同屏气泡数上限」

> 这些都经过核实：要么运行时零引用，要么当前游戏数据里一处没用到旧格式。

## 别慌：这些"危险区"其实没事（逐条对代码核实过）

早期文档（照一份旧快照写的）把下面这些列成"会丢数据"。**逐条查过编辑器与运行时源码后，确认它们都不成立**——留在这里，免得你还照着旧说法提心吊胆：

| 曾经的说法 | 实情 |
|---|---|
| 热区 `data`、观察说明文字保存会丢 | **不丢**。编辑器保留它不编辑的键；观察说明也原样带过 |
| 出生点只留坐标、别的丢 | **不丢**，同样保留未知键 |
| NPC 巡逻路线丢字段 | **不丢**。巡逻本来就只有路线/速度/移动动画三项，没有别的可丢 |
| 音频保存丢音量、循环 | **不丢**。保存时复制原条目全部字段，只更新音源路径 |
| 物品动态描述只能加、不能删 | **能删**，每条都有删除按钮 |
| 文本库不能删键 / 删分类 | **能删**，按钮 / 右键 / Delete 键都行 |
| 档案的页不能删、印象不能删、切走丢改动 | **都能删**；切换条目有自动提交保护，不丢 |
| 剧本阶段的结局标记保存会丢 | **不丢**（有专门防丢代码 + 回归测试）；只有给阶段改名才不继承 |
| 不能新建整张场景 | **能**，有「+新建场景」按钮 |
| 图对话节点、过场步骤重建会丢字段 | **不丢**。已知类型的全部字段都会往返保留，未知类型整块原样透传 |

## 动手前速查

| 我要… | 实情 |
|---|---|
| 改热区观察说明 / 出生点 / 巡逻 | 放心改，不丢 |
| 删物品动态描述 / 文本库键 / 档案页 | 放心删，有按钮 |
| 改场景多层背景、景深遮挡 | 主面板够不着，用 [场景深度工具](../editors/render-domain/scene-depth-editor) |
| 改玩家化身细节 | 去 [玩家化身面板](../editors/panels/avatar) |
| 改旗标迁移块 / 像素密度匹配 | 暂无界面，手改或找维护者 |
| 把区域改成深度地板 | 会清掉它的进入/停留/离开动作，想好再切 |
