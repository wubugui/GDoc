# 危险区速查

编辑器保存很稳：你在面板里填的东西会存住，你没碰的字段也不会被清掉。真正需要留意的只有下面**三类**。

:::tip[一句话准则]
面板里**有**的，放心填放心存；面板里**没有**的那几个活字段（见下方「盲区」），换专项工具或手改；保存时被清掉的字段，都是**废弃的旧字段**，清掉不影响游戏。
:::

---

## 一、盲区：这几处得换工具改

这些字段游戏运行时会用到，但**主编辑器对应面板没有入口**。编辑器不会替你丢掉它们，只是你在这个面板里改不了：

| 要改的东西 | 去哪改 |
|---|---|
| 场景**多层背景**、深度 / 遮挡碰撞主体 | **[场景深度工具](../editors/render-domain/scene-depth-editor)** |
| **玩家化身**外观细节 | **[玩家化身面板](../editors/panels/avatar)** |
| 旗标的**「迁移」「运行时」块** | 暂无界面，手改或找维护者 |
| 实体像素密度匹配 | 暂无界面，手改 |

:::warning[一个真隐患]
场景面板的**「重新导入背景」按钮**：如果一张场景本来有多层背景，用这个按钮换图会把它**压成单层**。多层背景一律走 [场景深度工具](../editors/render-domain/scene-depth-editor)，别用主面板的重导。
:::

---

## 二、这几个操作，点之前想清楚

不是悄悄发生的，是你**主动做了某个动作**才会有的后果：

- **把区域从「标准」改成「深度地板」** → 会清掉它的进入 / 停留 / 离开动作（还有气味）。深度地板区本来就不触发这些，正常用没问题；但**切过去再切回来，原来的动作找不回**。
- **改滤镜 id、位面「现世」的 id** → 只读，改名等于删掉重建。
- **删小游戏实例** → 登记表里没了，但**磁盘上那个旧实例文件不会自动删**，得手动清。
- **水域小游戏**的显示尺寸 / 命中半径**留空** → 表示用品类默认，保存时不写这两个键。

---

## 三、这些字段保存时会自动清掉——正常，不用管

保存时编辑器会顺手删掉几个**旧格式字段**，游戏运行时早就不用它们了。清掉是让数据变干净，不影响任何东西：

- NPC 的旧对白引用、区域旧矩形坐标与规矩槽位
- 任务的旧式「单一下一任务 id」
- 过场顶层的旧「命令列表」（现在用「步骤」）
- 规矩的几个旧字段、转盘一个已废弃的「同屏气泡数上限」

## 动手前速查

| 我要… | 怎么办 |
|---|---|
| 改热区观察说明 / 出生点 / 巡逻 | 放心改 |
| 删物品动态描述 / 文本库键 / 档案页 | 放心删，面板里都有按钮 |
| 改场景多层背景、景深遮挡 | 用 [场景深度工具](../editors/render-domain/scene-depth-editor) |
| 改玩家化身细节 | 去 [玩家化身面板](../editors/panels/avatar) |
| 改旗标迁移块 / 像素密度匹配 | 暂无界面，手改或找维护者 |
| 把区域改成深度地板 | 会清掉它的进入/停留/离开动作，想好再切 |
