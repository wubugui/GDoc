快捷键

以下以 主编辑器 为主；macOS 上 Ctrl 通常对应 Cmd，以编辑器菜单栏显示为准。

详解见 主编辑器怎么逛、运行预览。

保存​

快捷键作用
Ctrl+S保存当前面板
Ctrl+Shift+S保存全部已改面板

保存前若条目落在重建区，只保留面板里填过的项——见 危险区。

撤销与重做​

快捷键作用
Ctrl+Z撤销（当前面板内）
Ctrl+Y 或 Ctrl+Shift+Z重做

切到别的面板再回来，各面板各自记撤销栈。

面板导航（像浏览器翻页）​

快捷键作用
Alt+←后退到上一个打开过的面板
Alt+→前进

典型：场景摆好 NPC → 图对话改词 → Alt+← 回场景对位置。

运行预览​

快捷键作用
F5运行（内嵌）：先自动保存，再在「运行与预览 → Game」里起游戏
Ctrl+F5独立窗口预览（双屏对照）
Shift+F5停止正在跑的游戏

其它​

操作说明
左侧导航树过滤框输入关键词快速定位面板（如「任务」「音频」）
菜单 视图字号、布局等——以编辑器菜单为准
菜单 文件另存、校验等全局项——以编辑器菜单为准

图对话、过场等独立窗口工具的快捷键若与主编辑器不同，以该窗口菜单为准。

推荐日常循环​

  1. 改内容 → Ctrl+S
  2. F5 预览验证
  3. 不对 → Shift+F5 停 → Alt+← 回编辑面板 → 再改 → F5

常见问题​

为什么 Ctrl+S 按了没反应？ 先确认焦点在某个面板的编辑区里而不是左侧导航树——导航树聚焦时快捷键可能不生效。点一下面板正文再试。

保存快捷键和菜单里的「保存」按钮效果一样吗？ 一样，Ctrl+S/Ctrl+Shift+S 只是菜单「文件 → 保存」对应操作的快捷方式，不会比点按钮多做或少做什么。

为什么撤销跨面板没用？ 撤销栈按面板分开记——你在场景面板做的改动，切到图对话面板后再按 Ctrl+Z，撤的是图对话自己的历史，不会跳回场景那边。想撤场景的改动，得先切回场景面板。

独立窗口工具（图对话、过场等）能用这页的快捷键吗？ 不一定。这些是单独的窗口，保存/撤销等快捷键以它们自己菜单栏显示的为准，跟主编辑器不一定完全一致。

macOS 和 Windows 按键不一样怎么办？ 这页写的是常见约定（Ctrl），macOS 上一般对应 Cmd；具体以你当前打开的编辑器菜单栏上实际显示的按键为准，不同版本可能有出入。
