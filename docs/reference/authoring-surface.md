# 可编辑内容面

本页已迁至 **[危险区：逐面板后果清单](./danger-zone)**，这里只留一句话摘要和旧链接跳转，别把这页收藏成正文来看。

## 一句话摘要

主编辑器每块面板里的内容，填了就稳稳存住；连它不编辑的字段，多数地方也会原样保留。

真正需要留意的只有一小类**盲区**——游戏认这个字段，但面板没给入口，得换专项工具改（盲区字段**不会丢**，只是编辑不到）；外加几个你主动点了才触发的破坏性操作，和保存时自动清理的废弃旧字段。

逐面板「能改什么 / 要当心什么 / 哪里够不着」的完整清单，都在 **[危险区速查](./danger-zone)** 那一页，本页不重复贴一遍，以免两边改动脱节。
