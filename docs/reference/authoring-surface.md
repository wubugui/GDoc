# 可编辑内容面

本页已迁至 **[危险区：逐面板后果清单](./danger-zone)**，这里只留一句话摘要和旧链接跳转，别把这页收藏成正文来看。

## 一句话摘要

主编辑器每块面板里的内容，只分三种：

- **安全区**——面板上有格子填，填了就稳稳存住。
- **重建区**——面板保存时按自己认识的格子整段重写，你多写的、面板没提供输入框的内容会跟着消失。
- **盲区**——游戏认这个字段，但面板压根没给你摸得到的入口，得换专项工具或者先别碰。

逐面板「能改什么 / 哪里会丢 / 哪里够不着」的完整清单，都在 **[危险区：逐面板后果清单](./danger-zone)** 那一页，本页不重复贴一遍，以免两边改动脱节。
