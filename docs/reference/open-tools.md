# 工具打开方式

只回答：**我想用某件工具，该怎么打开？** 在游戏仓库根目录执行下列命令，或点对应按钮。

更完整的用途说明见 **[工具速查表](../editors/tool-matrix)**；概念说明见 **[编辑器 · 工具打开方式](../editors/launch-architecture)**。

---

## 两扇常用门

| 工具 | 命令 |
|---|---|
| **主编辑器**（改内容） | `./dev.sh editor` |
| **Web 控制台**（工具箱） | `./dev.sh console` |

控制台里点按钮，效果等同下表对应命令。

---

| `editor` | 主编辑器 |

| `asset-ingest` | 资源入库 |
| `image-resizer` | 图片缩放 |
| `filter-tool` | 滤镜工具 |
| `lightvol` | 光照体积实验室 |
| `anim-preview` | 动画预览 |
| `parallax-editor` | 视差编辑器 |
| `chronicle-sim-v2` | 编年史模拟 v2 |
| `chronicle-sim` | 编年史模拟 v3 |
| `skill-governance` | 治理台 |
| `validate-data` | 数据校验 |

## 其它启动方式

| 工具 | 怎么开 |
|---|---|
| 视频转图集 | 主编辑器菜单 **工具 → 外部工具 → 视频转图集** |
| 场景深度 | 主编辑器菜单 **工具 → 外部工具 → 场景深度** |
| 通用图编辑器 | 主编辑器菜单 **工具 → 外部工具 → 通用图编辑器** |
| 文案管理 | 主编辑器菜单 **工具 → 外部工具 → 文案管理** |
| 叙事状态机 | 主编辑器内嵌 Web 页，或 `npm run dev:narrative-editor` |
| 动画后处理 | 命令行批处理（无 GUI） |
| JSON 语言服务 | `./dev.sh json-lang` |

未注册到 `dev.sh` 的项，以 **[工具速查表](../editors/tool-matrix)** 该工具行的「启动」列为准。

## 游戏、构建、测试

| 你想… | 怎么开 |
|---|---|
| 主编辑器内嵌预览 | 主编辑器 `F5`，或左侧 **运行与预览 → Game** |
| 独立窗口预览 | `Ctrl+F5` |
| 浏览器起游戏 | `./dev.sh game start` 或控制台 **游戏 dev 开关** |
| 停游戏 | `./dev.sh game stop` 或控制台关闭 |
| 打正式包 | Web 控制台 **构建** |
| 跑自动化测试 | Web 控制台 **测试** |

## Web 控制台按钮

与上表 `./dev.sh` 任务名一一对应，另提供：

- **游戏 dev 开关**
- **构建** / **测试**
- **场景快捷入口**（跳进指定场景试玩）
- **叙事 warp 入口**（开发用快速跳转）

用法见 **[Web 控制台](../editors/concepts/web-console)**。

## 打不开？

| 现象 | 怎么办 |
|---|---|
| Python 环境缺失 | 仓库根目录跑 `./bootstrap.sh` |
| 任务名报错 | 对照上表，注意连字符（如 `asset-ingest`） |
| Web 工具起不来 | 确认 Node ≥ 20，看终端实际端口 |
| 点了没反应 | 看终端报错；关掉旧窗口再开 |

更多 → **[出问题怎么办](../tutorials/troubleshooting)**。
