编辑器手册
一盏油灯,一方书案 —— GameDraft 的一整套工具,就是你编纂这座游戏的笔与墨。

编辑器总览

做 GameDraft 的内容，就像在雾津的书案上编一本册子：场景、对白、过场、任务、物品、规矩、小游戏……每一样都有一件趁手的工具。整套工具有 21 件，听着多，但你平时只需要记住两扇门。

先认两扇门​

门什么时候开一句话
🏮 主编辑器要改游戏内容你的主工作台，游戏里几乎所有东西都在这里编——场景、对白、任务、物品、规矩、小游戏。
🧭 Web 控制台不知道该用哪个工具一块仪表盘，列了一排工具按钮，还能一键跑游戏、构建、跑测试。当"工具箱"翻。
记不住就记这句

改东西 → 开主编辑器。犯迷糊 → 开 Web 控制台。 其余工具，需要时再从这两扇门里找。

想直接翻某件工具​

按"用来做什么"分门别类，点名字进详情：

  • 在主编辑器里（一个窗口装下大部分活）：场景、叙事、规则经济、注册表、资源本地化——共 30 块面板，外加三件贯穿始终的顺手控件。
  • 素材与动画：asset_browser · asset_ingest · image_resizer · video_to_atlas · animation_pipeline · anim_preview
  • 对话与叙事：dialogue_graph_editor · graph_editor · narrative_editor_web · copy_manager
  • 画面与光影：scene_depth_editor · lightvolume_lab · parallax_editor · filter_tool
  • 叙事模拟：chronicle_sim_v2 · chronicle_sim_v3
  • 服务：json_lang · skill_workflow_governance
  • 管理台：生产工作台

每件工具都有独立一页：做什么、怎么开、有什么要当心的。完整清单和启动命令见 工具速查表。

动手前，先记一条保命规矩​

主编辑器保存时，有些地方会被它整体重写——你在文件里手写塞的东西，开一次那个面板再保存，就没了。这叫 危险区。

第一次做内容，务必先花两分钟看这页：可编辑内容面 / 危险区——它会告诉你哪里随便改都安全、哪里一保存就丢、哪里编辑器根本够不着。
