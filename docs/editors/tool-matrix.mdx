# 工具速查表

GameDraft 全部 **<ToolCount /> 个编辑器/工具** 按域分组。表内 **工具名可点进用法手册**；「怎么开」列给出 `./dev.sh <任务名>`、**主编辑器 → 外部工具** 或 **Web 控制台** 按钮。

## 统一怎么开

| 你想… | 做法 |
|---|---|
| 改对白、场景、任务 | `./dev.sh editor` 开 **主编辑器** |
| 不确定用哪个工具 | `./dev.sh console` 开 **Web 控制台**，点按钮 |
| 表里写了 `./dev.sh xxx` | 在仓库根目录执行该行 |
| 表里写了「主编辑器 → 外部工具」 | 先开主编辑器，菜单 **工具 → 外部工具** |
| 叙事状态机 | 主编辑器 **叙事编排 → 叙事状态机**（不必单独起服） |
