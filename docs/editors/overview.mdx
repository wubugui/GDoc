# 编辑器总览

做 GameDraft 的内容，就像在雾津的书案上编一本册子：场景、对白、过场、任务、物品、规矩、小游戏……每一样都有一件趁手的工具。整套工具有 **<ToolCount /> 件**，听着多，但你**平时只需要记住两扇门**。

| 门 | 什么时候开 | 一句话 |
|---|---|---|
| **[🏮 主编辑器](./main-editor/overview)** | 要**改游戏内容** | 你的主工作台，游戏里几乎所有东西都在这里编——场景、对白、任务、物品、规矩、小游戏。 |
| **[🧭 Web 控制台](./web-console)** | **不知道该用哪个工具** | 一块仪表盘，列了一排工具按钮，还能一键跑游戏、构建、跑测试。当"工具箱"翻。 |

:::tip[记不住就记这句]
**改东西 → 开主编辑器。犯迷糊 → 开 Web 控制台。** 其余工具，需要时再从这两扇门里找。
:::

## 动手前，先记一条保命规矩

主编辑器保存时，有些地方会被它**整体重写**——你在文件里手写塞的东西，开一次那个面板再保存，就没了。这叫 **危险区**。

第一次做内容，务必先花两分钟看这页：**[可编辑内容面 / 危险区](../reference/authoring-surface)**——它会告诉你哪里随便改都安全、哪里一保存就丢、哪里编辑器根本够不着。
