# 编辑器总览

做 GameDraft 的内容，就像在雾津的书案上编一本册子：场景、对白、过场、任务、物品、规矩、小游戏……每一样都有一件趁手的工具。整套工具有 **<ToolCount /> 件**，听着多，但你**平时只需要记住两扇门**。

| 门 | 什么时候开 | 一句话 |
|---|---|---|
| **[主编辑器](./main-editor/overview)** | 要**改游戏内容** | 主工作台，30 块面板 + 运行预览，游戏里几乎所有内容在这里编。 |
| **[Web 控制台](./concepts/web-console)** | **不知道该用哪个工具** | 浏览器仪表盘，列工具按钮，还能一键跑游戏、构建、测试。 |

:::tip[记不住就记这句]
**改东西 → 开主编辑器。犯迷糊 → 开 Web 控制台。** 其余工具，需要时再从这两扇门里找。
:::

## 动手前，先记一条保命规矩

主编辑器保存时，有些地方会被**整体重写**——你在界面之外多写的内容，开面板再保存可能就没了。这叫 **[危险区](./concepts/danger-zone)**。

第一次做内容，务必先花两分钟读完；逐面板细节查 **[可编辑面参考](../reference/authoring-surface)**。
