# 放一个会说话的 NPC

雾津的码头、茶馆、义庄——少了会搭话的 NPC，就像少了人声。这一页教你在场景里**放一个 NPC**，绑上一张**对白图**，让玩家走近按互动键就能开口。

---

## 读完你能做到什么

- 在场景画布上新增 NPC 并拖到合适位置
- 给 NPC 起名、绑对白图与入口节点
- 设互动距离与面向
- 运行预览里走过去跟 NPC 说话

---

## 怎么开工具

主编辑器 → **物理世界 → 场景**（NPC 是场景的一部分）

对白内容在 **叙事编排 → 图对话** 里编。两扇面板常来回切。

```bash
./dev.sh editor
```

---

## 逐步操作

### 第 1 步：准备对白（若还没有）

1. 打开 **图对话**
2. 选或新建一张对白图（见 [改你的第一句对白](./first-line)）
3. 至少放：**台词节点** → **结束**
4. 记住图的名字，以及第一个**台词节点**的入口名（检查器里「入口」或列表第一项）

### 第 2 步：打开目标场景

1. **场景**面板 → 场景列表选「雾津街头」等
2. 画布显示当前场景

### 第 3 步：新增 NPC

1. 列表面板或工具栏点 **新增 NPC**
2. 画布上出现 NPC 标记，拖到要站的位置——比如关二狗蹲在茶馆后门
3. 右侧检查器填写：

| 项 | 说明 |
|---|---|
| **标识** | 系统用，英文或拼音，别跟别的 NPC 撞名 |
| **显示名** | 玩家看到的名字，如「关二狗」 |
| **对白图** | 下拉选刚准备好的那张图 |
| **对白入口** | 从这张图的哪个节点开始聊 |
| **互动距离** | 玩家多近能按 E 说话 |
| **初始面向** | 左/右，决定立绘或精灵朝向 |
| **镜头缩放** | 对话时镜头推近程度（可选） |

### 第 4 步：动画与碰撞（可选）

- **动画文件 / 初始动作**：让 NPC 站着别发呆（动画在资源管线里备好）
- **碰撞多边形**：在画布上拖点，框住可走/不可走的边界

若 NPC 只在特定任务阶段出现，在检查器里加**条件**；条件用法见 [怎么设条件](../editors/concepts/conditions)。

### 第 5 步：保存并验证

1. **Ctrl+S** 保存场景
2. **F5** 运行预览
3. 控制关二狗（或玩家）走到 NPC 旁，按 **E** 或 **空格**
4. 对白框应弹出，台词来自你绑的对白图

## 另一种做法：用热区挂 NPC

场景里还有一种 **NPC 型热区**——不占独立 NPC 槽，适合「调查点顺便聊两句」。做法：

1. 场景里 **新增热区**，类型选 **NPC 对话**
2. 指定引用哪个已登记的 NPC 身份
3. 配动作或对白图（视检查器提供的模式）

独立 **NPC 实体**适合长期站在地图上的角色；**热区型**适合一次性或跟物件绑定的对话。详见 [场景面板](../editors/panels/scene)。

## 雾津小例子

**任务**：在雾津街头放李天狗，玩家一靠近就能问寻狗的事。

1. 图对话里备好「李天狗_问路」对白图，入口是第一句「道士穷，狗也穷？」
2. 场景选雾津街头，新增 NPC，显示名「李天狗」
3. 绑对白图「李天狗_问路」，入口选默认第一句
4. 拖到土地庙台阶旁，互动距离略大一点（玩家不用贴脸）
5. **F5** 走过去搭话——雾里多一个能回嘴的道士
