做一个视差过场

有些戏不宜让玩家走着看——进城隍庙前，远山、牌楼、人影宜分层滑过，像卷轴徐徐展开。视差过场干的就是这个：多层图各走各的速度，镜头推移时远近错落。这一页从摆图层到挂进过场，走通一遍。

读完你能做到什么​

  • 用视差编辑器建多层场景、设关键帧与运动轨迹
  • 保存后供过场调用
  • 预览里确认图层速度、时长、衔接
  • 在游戏里播一遍完整过场

视差是什么​

视差：近处动得快、远处动得慢，眼睛觉得有纵深。过场里用多层静态或动态图，按不同速率平移，营造「镜头在扫街」的效果。

视差数据和普通场景是分开的——在视差编辑器里编，在过场面板里用「播放视差场景」这类步骤触发。

第 1 步：打开视差编辑器​

./dev.sh parallax-editor

浏览器会打开编辑界面（本地端口由工具提示）。渲染和游戏一致，调完所见即所得。

也可 ./dev.sh editor → 工具 → 外部工具 → 视差场景编辑器。

第 2 步：建场景、摆图层​

  1. 新建或打开一个视差场景——例如 城隍庙 entrada（进庙前外景）
  2. 加图层（从下到上）：
    • 最远：雾津远山、天色
    • 中间：牌楼、树影
    • 近处：石阶、灯笼、行人剪影
  3. 每层设视差速率——远景 0.2、近景 1.0 是常见起点，按感觉微调
  4. 素材先用 导入一张素材 进库，编辑器里按短名选图

操作示意​

第 3 步：关键帧与轨迹​

  1. 在时间轴上打关键帧：镜头起止位置、时长
  2. 拖镜头路径——横向扫街、缓慢推进、停在庙门前均可
  3. 逐层检查：有没有穿帮、边缘是否露白（图不够宽要加长或循环）
  4. 保存——结果写入工程的视差场景库

第 4 步：挂进过场​

./dev.sh editor

叙事编排 → 过场：

  1. 新建或打开一段过场——如进庙前的引子
  2. 在步骤序列里加播放视差场景（名称以面板里为准）
  3. 选中刚编的视差场景 id
  4. 后面可接旁白、切场景、黑场等步骤

过场面板里也有按钮可直达视差编辑器改同一景。

第 5 步：运行预览​

F5 触发这段过场：

  • 各层速度是否自然
  • 总时长和旁白、音乐是否合拍
  • 播完是否顺利切到下一个场景或步骤

流程示意​

进阶：把视差摸透​

入门流程只带你做完一段最简单的视差。真做起雾津的过场，这几件事早晚要碰上：

图层怎么分不穿帮​

  • 图要够宽：镜头要横移多远，图层就要覆盖多远的范围，速率越慢的层相对移动量越小，但也不能刚好卡边——留够余量，不然运镜到头会露白边。
  • 循环平铺 vs 一次性长图：远景（雾、天色）适合做成能首尾相接的循环图，省资源；近景细节多的（石阶、灯笼）通常一次性长图更省心，别让接缝正好落在镜头会扫到的地方。
  • 图层顺序即渲染顺序——从下到上建，别把近景图层建在远景下面，不然会被挡住。

轨迹怎么编排​

常见几种运镜，可以单独用也可以接起来组合：

轨迹类型效果适合场景
横向扫街镜头平移，像卷轴展开进城、逛街、赶路
缓慢推进镜头从远景推向近景靠近一处建筑或人物
静止定格镜头不动，只让图层自己缓慢漂移单纯氛围镜头，比如夜色降临
组合运镜先横移，再在终点推近停住「一路走到庙门前抬头看匾」这类叙事镜头

关键帧不宜太密——太密显得镜头在抖，一般3～5 个关键点（起点、一两个过渡点、终点）就够表达一段情绪。透明度和缩放的关键帧还能单独用来做「淡入淡出」——比如让某层雾气从无到有渐渐浓起来，不一定非要跟位移绑在一起。

复用与效率​

  • 同一个视差场景 id，可以被多段过场共同引用——比如「进城隍庙」和「离开城隍庙回望」若用同一批图层，改一次镜头方向就能派两个过场用，不用重做图层。
  • 老手习惯：编完先切 Ctrl+F5 独立窗口整屏看一遍，内嵌预览区域小，边缘穿帮和接缝容易漏看。
  • 和 滤镜、光照体积配合使用时，先把视差的运镜和时长定下来，再调滤镜色调——顺序反过来容易白改。

危险区与边界​

  • 视差场景是独立数据，不挂在场景或过场文件里——过场面板只存一个「指向哪个视差场景 id」的引用。改视差场景的 id 或删掉它，所有引用它的过场步骤都会跟着失效（游戏里播放时可能黑屏或报错）。改名前先想清楚有哪些过场在用。
  • 图层素材必须先 入库 才能在编辑器里选到；直接把文件丢进项目目录，编辑器认不到。
  • 编辑器未保存就关浏览器，当次改动会丢——和主编辑器的 Ctrl+S 习惯一样，编完随手保存。
  • 更完整的「哪里改了会丢、哪里编辑器够不到」的全局说明，见 危险区。

雾津小例子​

李天狗带关二狗头一回见城隍庙，要一层层揭开牌楼：

  1. 视差编辑器建 chenghuang_approach，四层：雾、牌楼、街人、石阶灯笼
  2. 镜头从远推近，四秒停庙门匾下
  3. 过场 intro_chenghuang 第一步播该视差，第二步接李天狗旁白
  4. F5 从新档或调试入口播过场，听旁白是否踩准画面停住那一刻

卷轴拉开，庙才显灵。

常见问题​

现象原因怎么办
打开视差编辑器一片空白端口被占用、复用了旧实例看终端提示的实际端口；关掉旧窗口重开
游戏里图层位置和编辑器里对不上场景/过场里的镜头缩放、分辨率设置不同用同一套预览分辨率核对，别只看编辑器画布
过场播到这步黑屏视差场景 id 填错，或该 id 已被删/改名回过场步骤核对 id，和视差编辑器列表逐字比对
边缘露白、穿帮图层不够宽，镜头扫到了图外加长图或改成循环平铺
多层看起来像同一速度在动视差系数没拉开差距远景调小、近景调大，差距明显才有纵深感
预览很卡图层过多或分辨率过大精简图层数量，远景可适当降分辨率
